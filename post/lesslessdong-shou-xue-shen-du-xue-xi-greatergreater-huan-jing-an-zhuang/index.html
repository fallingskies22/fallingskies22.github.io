《动手学深度学习》- 环境安装

2025-04-01

前言

https://zh.d2l.ai/chapter_installation/index.html
https://www.zhihu.com/zvideo/1363284223420436480

步骤

安装python

为了匹配教程环境
安装python3.8.5
https://www.python.org/downloads/release/python-385/

安装miniconda

https://repo.anaconda.com/miniconda/Miniconda3-py38_4.9.2-Windows-x86_64.exe

任务栏搜索Anaconda Prompt

打开即可执行命令，如conda list。

创建一个新的环境

conda create --name d2l python=3.8.5 -y

激活环境

conda activate d2l

(注意：后续命令都在conda中执行)

安装深度学习框架和d2l软件

CPU版本pytorch

pip install torch==1.12.0
pip install torchvision==0.13.0

pip install d2l==0.17.6

下载d2l需要安装rust环境
参考https://blog.csdn.net/shi_yu123456/article/details/146301425

下载报错ERROR: Failed building wheel for pywinpty，需安装C++构建工具。
参考https://blog.csdn.net/qq_40682931/article/details/145815327

GPU版本pytorch

使用

nvidia-smi

查看显卡驱动版本

在https://pytorch.org/安装对应版本的包

pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

下载D2L Notebook

https://zh-v2.d2l.ai/d2l-zh-2.0.0.zip

访问notebook

jupyter notebook

需要切换到下载文件目录
使用cd ..切换到上层目录，只有切换到最上层才能切换磁盘。
